# Sinnoh Trainer Hub Template

A bold, colorful Pokémon Sinnoh-themed landing page template featuring legendary Pokémon backgrounds.

## Preview

Open `index.html` in your browser to see the template.

## Features

- Dark Sinnoh region theme with icy blue gradients
- Animated legendary Pokémon backgrounds (Dialga, Palkia, Giratina, Arceus, Garchomp)
- Falling snow particles effect
- Fully responsive design
- Smooth scroll animations
- No dependencies - pure HTML/CSS/JS

## Sections

1. **Navbar** - Fixed navigation with scroll effect
2. **Hero** - Main headline with legendary Pokémon backdrop
3. **Features** - 6 feature cards with icons
4. **Pokédex** - Showcase popular Sinnoh Pokémon
5. **Metrics** - Key statistics
6. **Pricing** - 3-tier pricing table
7. **FAQ** - Accordion FAQ section
8. **CTA** - Call-to-action banner
9. **Footer** - Links and branding

## Customization

### Colors
Edit CSS variables in `style.css`:
```css
:root {
    --primary: #3b82f6;
    --accent-ice: #06b6d4;
    --accent-dragon: #8b5cf6;
    --bg: #0a0f1a;
}
```

### Images
Replace PNG files in the directory:
- `dialga.png` - Left background
- `palkia.png` - Right background
- `giratina.png` - Center-left
- `arceus.png` - Center-right
- `garchomp.png` - Bottom center
