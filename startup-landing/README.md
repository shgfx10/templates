# Nexus - Startup Landing Page Template

A bold, colorful landing page template designed for tech startups.

## Features

- Modern dark theme with gradient accents
- Fully responsive (mobile, tablet, desktop)
- Smooth scroll animations
- No dependencies - pure HTML/CSS/JS
- Easy to customize

## Sections

1. **Navbar** - Fixed navigation with scroll effect
2. **Hero** - Main headline with animated gradient orbs
3. **Logo Cloud** - Trusted by companies section
4. **Features** - 6 feature cards with icons
5. **Metrics** - Key statistics showcase
6. **Pricing** - 3-tier pricing table
7. **FAQ** - Accordion FAQ section
8. **CTA** - Call-to-action banner
9. **Footer** - Links and branding

## Customization

### Colors
Edit the CSS variables in `style.css`:
```css
:root {
    --primary: #6c5ce7;      /* Main brand color */
    --accent-1: #00cec9;     /* Secondary accent */
    --accent-2: #fd79a8;     /* Pink accent */
    --bg: #0a0a0f;           /* Background */
    --surface: #12121a;      /* Card background */
}
```

### Content
- Replace "Nexus" with your brand name
- Update pricing values
- Replace placeholder images (avatars, logos)
- Edit copy text in `index.html`
